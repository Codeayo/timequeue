# DineQueue

Project

DineQueue is an appointment booking system with waitlist auto-fill functionality.

## Features

- User registration and login
- Host creates appointment slots
- Users book available slots
- Waitlist system when slots are full
- Automatic promotion from waitlist when a booking is canceled

## Tech Stack

Frontend:
- Vue
- Vite

Backend:
- Node.js
- Express

Database:
- SQLite
